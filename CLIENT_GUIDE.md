# HR-бот «Ченсон» — руководство для клиента

Telegram-бот для массового подбора персонала в сеть корейских кафе «Ченсон»
(Ташкент, филиалы: Куйлюк, Чиланзар, Сайрам, Сеул Мун). Бот сам собирает
анкеты, отсеивает неподходящих по требованиям компании, раздаёт карточки
кандидатов в чаты филиалов и ведёт всю воронку — от отклика до выхода на
работу.

---

## 1. Путь кандидата

Кандидат открывает бота по ссылке из объявления и видит меню:
«Откликнуться», «Список вакансий», «Условия работы».

1. Выбирает вакансию из **открытых** (позиция + филиал).
2. Заполняет анкету (бот спрашивает по одному вопросу, есть кнопка «Назад»):
   ФИО, возраст, пол, телефон (кнопка «Поделиться контактом»), район, опыт
   в общепите (у поваров — стаж в годах и специализация: горячий/холодный
   цех, суши, универсал), знание русского, готовность работать со свининой
   и алкоголем, для мужчин — наличие военного билета/приписного.
3. **Автоскрининг** мгновенно проверяет анкету по требованиям компании.
   - Не подходит → вежливый отказ со ссылкой на «требования компании»,
     без обидных формулировок. Причина (возраст/опыт/язык/военный
     билет/свинина-алкоголь) сохраняется только для внутренней аналитики.
   - Подходит → бот показывает условия вакансии и адрес собеседования
     и спрашивает: «Подтверждаете, что придёте?»
4. «Да» → карточка кандидата улетает менеджеру филиала и рекрутеру.
   «Нет» → бот предлагает другие вакансии.

**Повара** — особый случай: куда бы повар ни устраивался, собеседование
всегда на Куйлюке у шеф-повара. Бот сам подставляет правильный адрес.

Повторный отклик возможен только после завершения предыдущего.

## 2. Карточки в чаты филиалов

Как только кандидат подтвердил собеседование, его карточка (ФИО, возраст,
пол, вакансия, опыт, район, телефон, username) приходит:

- в Telegram-чат соответствующего филиала;
- поварские — в чат «Кухня» и лично шеф-повару;
- дубль — рекрутеру.

Персональные данные появляются **только в служебных чатах**.

## 3. Менеджер филиала и шеф-повар

У менеджера своё меню в боте:

- **Подать заявку на персонал** — «нужно 2 бармена, вечерняя смена» →
  заявка уходит рекрутеру, тот в один клик превращает её в вакансию,
  задаёт уточняющий вопрос или отклоняет.
- **Открытые вакансии моего филиала** — с прогрессом закрытия (нанято 2 из 5).
- **Мои кандидаты** — все, кто подтвердил собеседование. Менеджер сам
  отмечает: «Пришёл», «Не пришёл», «Вышел на стажировку», «Не прошёл
  стажировку» (с причиной), «Принят». Каждое изменение дублируется рекрутеру.

Менеджер видит **только свой филиал**. Шеф-повар видит поваров всех филиалов.

## 4. Рекрутер

- Управление вакансиями: создание (зарплата, график, описание, квота),
  закрытие, изменение квот.
- Готовые **шаблоны вакансий** для копирования в Telegram-каналы и HH
  + ссылка на папку с медиа.
- Кандидаты: фильтры по филиалу/позиции/статусу, полная анкета, смена
  статусов (отказ работодателя — только с причиной).
- **Отчёт за период**: сколько откликнулось → прошли скрининг → пришли →
  вышли на стажировку → приняты (с конверсией каждого этапа), причины
  отказов (отдельно видно «свинина/алкоголь»), эффективность источников.
  Кнопка «Скачать Excel» — все анкеты периода, включая отказы.
- **День сверки**: каждый понедельник в 10:00 бот сам присылает чеклист —
  какие вакансии актуальны, кто вышел, кто слетел со стажировки, на чём
  теряем кандидатов.

## 5. Источники трафика

Для каждого канала — своя ссылка: `t.me/<бот>?start=hh`,
`?start=tg_kanal`, `?start=olx`, `?start=qr`. В отчётах видно, сколько
кандидатов принёс каждый канал.

## 6. Требования к кандидатам — настраиваются без программиста

Все пороги (возрастные вилки по позициям, отдельный потолок возраста для
женщин-поваров, минимальный стаж поваров) меняются командой в боте за
10 секунд. Ужесточили набор официантов до 25 лет? Одна команда — и скрининг
уже работает по-новому.

## 7. Безопасность

- Доступ к служебным функциям строго по списку Telegram ID.
- Телефоны и анкеты — только в служебных чатах.
- Все действия сотрудников логируются (кто сменил статус, кто создал
  вакансию, кто изменил настройки).

## 8. Что нужно для запуска

1. Токен бота от @BotFather (3 минуты, бот можно назвать и оформить
   под бренд «Ченсон»).
2. Telegram ID сотрудников (каждый получает свой через @userinfobot).
3. Добавить бота в рабочие чаты филиалов и чат кухни (привязка — одной
   командой в каждом чате).
4. VPS от ~$5/мес — бот работает 24/7, перезапускается сам.
